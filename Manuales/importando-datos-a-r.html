Capítulo 5 Importando datos a R

La forma más sencilla de incorporar datos a R es mediante el clásico copy/paste. Se utiliza la función read.delim() para incorporar al ambiente de R una base de datos. Primero se copia (Ctrl+c) el área de la hoja de cálculo que se quiere incorporar a R, sin cerrar la hoja de cálculo, ya que sólo funciona mientras la aplicación se encuentra utilizando la memoria, se incorpora a R con la función read.delim().

  ejemplo2 <- read.delim("clipboard")

corroboramos que lo tenemos en nuestro ambiente de trabajo escribiendo:

ejemplo2

El problema de esta forma de incorporar datos es que cada vez que queramos empezar de nuevo el análisis tendríamos que volver a copiar y pegar las partes a analizar. Esto implica que dentro de un tiempo no nos vamos a acordar de la hoja de cálculo que usamos para hacer el copy/paste, ni de qué parte de la hoja de cálculo usamos para hacer los análisis. Por esto, para automatizar los procesos, incorporamos la base de datos en formato ASCII con las funciones.

read.table()
read.csv()

Por ejemplo, queremos importar una base de datos

ejemplo1 <- read.table("~/ejemplo1.txt")

La función read.table() tiene una serie de argumentos, por ejemplo, si nuestra BD tiene encabezados o no.

ejemplo1 <- read.table("ejemplo1.txt", header = TRUE)
head(ejemplo1)
tail(ejemplo1)
ejemplo1 <- read.table("sunfleck.txt", header = TRUE)
ejemplo1

Si es una BD separado por comas, .csv usamos la función read.csv().

ejemplo <- read.csv("ejemplo1.csv", header=TRUE)
ejemplo <- read.csv("sunfleck.csv", header = TRUE)
ejemplo

5.1 Guardando datos

La base de datos se puede guardar en formato de R con la función save():

save(ejemplo1,file="ejemplo1.Rdata")

después se consulta si se ha incorporado al ambiente la base de datos escribiendo sunfleck. La base de datos se salva en formato R con la función save().

save(ejemplo,file="sunfleck.Rdata")
